# Modern Business Landing Page

A fully responsive, modern business landing page built with React, Tailwind CSS, and Framer Motion. Features dark mode support, smooth animations, and enhanced user experience.

## Features

- 🎨 Modern UI/UX with Tailwind CSS
- 🌓 Dark Mode Support
- 🎬 Smooth Page Transitions
- 📱 Fully Responsive Design
- 🎯 Performance Optimized
- ⚡ Fast Loading Times
- 🔄 Scroll Progress Indicator
- ⬆️ Scroll to Top Button
- 📝 Contact Form with Validation
- 🔔 Toast Notifications

## Tech Stack

- React
- Tailwind CSS
- Framer Motion
- React Router
- React Icons

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/glossary_tech.git
```

2. Install dependencies
```bash
cd glossary_tech
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Build for production
```bash
npm run build
```
